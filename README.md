## Requirements
- node v23.3.0
- Apex oracle api setup

## Installation
```bash
git clone https://github.com/tushyagupta81/dbmsProj ./dbms-project
cd dbms-project
npm i
npm run dev
```

>#### <span style="color:blue">NOTE</span>
>Apex API has been turned off
